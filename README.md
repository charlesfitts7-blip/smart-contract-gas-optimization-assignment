
# Yield Vault Gas Optimization Assignment

## Objective

Optimize the smart contracts for gas efficiency while preserving correct functionality.

## Steps

1. Install dependencies

npm install

2. Compile contracts

npx hardhat compile

3. Run tests

npx hardhat test

## Task

- Identify gas inefficiencies
- Optimize the contracts
- Ensure tests pass
- Create a branch with your name and submit a PR

Example branch:

git checkout -b candidate-yourname-gas-optimization
